# Projeto Kitty

Este é um projeto inicial e básico para uma aplicação web temática sobre gatos. Inclui arquivos HTML, CSS e JavaScript para uma página simples e interativa.

## Estrutura dos Arquivos
- `index.html`: Página principal da aplicação.
- `styles.css`: Estilos para layout e aparência.
- `script.js`: Funcionalidades JavaScript básicas.
- `README.md`: Este arquivo de documentação.

## Como Usar
1. Abra o arquivo `index.html` em um navegador web.
2. Clique no botão para ver uma mensagem surpresa.
