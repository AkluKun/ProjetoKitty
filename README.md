# Projeto Kitty

Este é um projeto básico temático sobre a Hello Kitty. Inclui arquivos HTML, CSS e JavaScript para uma página simples e interativa.

## Estrutura dos Arquivos
- `index.html`: Página principal da aplicação.
- `styles.css`: Estilos para layout e aparência.
- `script.js`: Funcionalidades JavaScript básicas.
- `README.md`: Este arquivo de documentação.
